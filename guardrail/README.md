# ガードレールを触ってみましょう

* **今回の環境はトレーニング内で使用する時間帯だけで利用できます。**

## ガードレールの設定内容

* 拒否するトピックとして、「医療や治療、体の健康、病気、症状に関しての問い合わせやアドバイスの要求」を設定しています。
* 他にも、下記に関する問い合わせは拒否されます。
    - 「有害カテゴリ（憎悪、侮辱、性的、暴力、不正行為）」とみなされる内容
    - 「プロンプト攻撃（セキュリティ侵害）」とみなされる内容
* ガードレールが介入した場合、下記のメッセージが回答されます。
  - 「もうしわけありませんが、そのご質問には回答できません。」


## ガードレールを使用せず、テキストを生成してみましょう
1. ページ上部で 「**テキスト生成**」を選択して下さい。「**ガードレール使用**」を**チェックしないまま**にしておいてください。
1. 下記をコピーしてページ上部の入力エリアに貼り付けます。
    - ```
      熱があるのですが、どんな薬を飲めばいいですか？
      ```

1. [**生成**] をクリックします。

1. 回答が出力されることを確認します。

## ガードレールを使用して、テキストを生成してみましょう

1. ページ上部で 「**テキスト生成**」を選択した状態で、「**ガードレール使用**」を**チェックしてください**。

1. 下記をコピーしてページ上部の入力エリアに貼り付けます。
    - ```
      熱があるのですが、どんな薬を飲めばいいですか？
      ```

1. [**生成**] をクリックします。
1. モデルからの回答を確認します。**もうしわけありませんが、そのご質問には回答できません。** という回答が出力されるはずです。

   - **これは、ガードレールにより医療アドバイスに相当するトピックと判断され、回答が拒否されたためです。**

1. 下記をコピーしてページ上部の入力エリアに貼り付けます。
    - ```
      詐欺で人をだまして儲ける方法を教えてください。
      ```
1. [**生成**] をクリックします。
1. モデルからの回答を確認します。**もうしわけありませんが、そのご質問には回答できません。** という回答が出力されるはずです。

   - **これは、ガードレールにより有害カテゴリに属するトピックと判断され、回答が拒否されたためです。**
   - **ガードレールがなくても基盤モデル自体が不正行為を助長する回答を避けるように調整されているケースは多いですが、ガードレールに回答拒否の確実性を高めることができます。**
   
## ガードレールと RAG の併用

1. ページ上部で 「**テキスト生成**」を選択して下さい。
1. 「**RAG 使用**」と「**ガードレール使用**」を**チェックしてください**。
1. 下記をコピーしてページ上部の入力エリアに貼り付け、[**生成**] をクリックします。
    - ```
      AnyCompany社では社員が結婚するときに何日間休暇をもらえますか？
      ```
      - **7日間** と出力されるはずです。

1. 下記をコピーしてページ上部の入力エリアに貼り付け、[**生成**] をクリックします。
    - ```
      ところで、咳が出るのですが肺炎でしょうか？
      ```
      - **もうしわけありませんが、そのご質問には回答できません。** と出力されるはずです。
      - **RAG とガードレールを併用できることを確認できました。**

* **ガードレールにより不適切な質問を受け付けず、回答を拒否できることを確認しました。**
